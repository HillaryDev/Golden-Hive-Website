    Golden Hive — Honey Shop Website

A modern, responsive honey shop website built with HTML, CSS, and JavaScript. Golden Hive is a static front-end project showcasing a premium honey brand with an elegant design, product catalogue, and customer contact features.



    Live Demo

[View Live Site](https://hillarydev.github.io/golden-hive)


    Preview

[Golden Hive Preview](preview.png)


    Features

- Responsive design — works on mobile, tablet, and desktop
- Product catalogue — filterable honey product listings
- Smooth navigation — single-page layout with anchor links
- Social media links — Instagram, Facebook, TikTok, and WhatsApp click-to-chat
- Contact section — easy reach via WhatsApp with a pre-filled message
- Clean footer — branding, links, and social icons


 Built With

- HTML5 — page structure and content
- CSS3 — styling, layout (Flexbox), hover effects, and responsive design
- JavaScript — interactivity and filtering
- SVG Icons — social media icons via SVG Repo

     Getting Started

     View locally

1. Clone the repository:
```bash
git clone https://github.com/your-username/golden-hive.git
```

2. Navigate into the project folder:
```bash
cd golden-hive
```

3. Open `index.html` in your browser — no build tools or installs needed.


        Deployment

This site is deployed using GitHub Pages.

To deploy your own copy:
1. Fork this repository
2. Go to Settings → Pages
3. Set branch to `main` and folder to `/ (root)`
4. Your site will be live at `https://your-username.github.io/golden-hive`
